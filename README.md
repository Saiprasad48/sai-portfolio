# Sai Prasad Reddy Kukudala Portfolio

A modern one-page portfolio built with React and Vite.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## What to edit

- Update profile details in `src/data.js`
- Replace placeholder GitHub links for any projects whose repo URL is different
- Replace the resume in `public/Sai-Prasad-Reddy-Kukudala-Resume.pdf` whenever your resume changes
- Deploy using Vercel, Netlify, or GitHub Pages

## Suggested deployment

Vercel is the easiest option:

1. Push this folder to a GitHub repository named `sai-portfolio` or `portfolio`
2. Go to Vercel and import the GitHub repo
3. Keep the default Vite settings
4. Deploy

Then add the live portfolio link to GitHub, LinkedIn, and your resume.
